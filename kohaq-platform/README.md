# KOHAQ Platform

Tagline: NEW TOMORROW

This repository contains the KOHAQ platform scaffold built with Next.js 14,
TypeScript, Tailwind CSS v3, and a modular app structure for LMS, careers,
community, media, dashboards, and AI microservice integration.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Scripts

- npm run dev
- npm run build
- npm run start
- npm run lint
- npm run type-check
